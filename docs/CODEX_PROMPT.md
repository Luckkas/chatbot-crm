# Prompt inicial para usar no Codex

Leia todos os arquivos `.md` deste repositório antes de começar.

Quero criar um MVP/protótipo funcional de um painel para chatbot WhatsApp.

O projeto deve usar:

- Node.js
- Express
- Prisma ORM
- PostgreSQL local
- EJS
- Bootstrap

O objetivo principal é entregar rápido um painel funcional para testes e demonstração, não criar uma arquitetura excessivamente complexa.

## Primeira tarefa

Crie a estrutura inicial do projeto usando Node.js com Express, Prisma ORM, PostgreSQL, EJS e Bootstrap.

Não implemente autenticação ainda.

Crie:

- estrutura de pastas;
- configuração do Express;
- configuração do Prisma;
- schema Prisma inicial;
- rotas API;
- controllers;
- services;
- views EJS iniciais;
- dashboard inicial;
- seed de opções de menu.

Crie os models iniciais:

- Cliente
- Conversa
- Mensagem
- MenuOpcao
- LogIntegracao

Crie seed inicial com opções de menu:

1. Suporte
2. Financeiro
3. Planos
Inválido. Mensagem padrão para opção não reconhecida.

Depois de criar a estrutura, explique rapidamente como rodar o projeto e quais endpoints foram criados.
